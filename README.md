# Heart Disease Prediction using Machine Learning

## Overview

This project builds a machine learning model in R to predict the likelihood of heart disease using clinical patient data such as age, cholesterol, blood pressure, and heart rate.

## Problem Statement

Early detection of heart disease is critical in healthcare. This project aims to assist in identifying high-risk patients using data-driven predictive modeling.

## Tech Stack

- R
- R Markdown
- dplyr
- ggplot2
- caret
- pROC

## Workflow

1. Data Collection (UCI Heart Disease Dataset)
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Correlation Analysis
5. Model Training (Classification)
6. Model Evaluation (Accuracy, ROC Curve)

## Results

- Achieved classification-based prediction of heart disease risk
- Evaluated model using accuracy and ROC metrics
- Identified key health indicators influencing prediction

The model evaluates patient risk using classification techniques and performance metrics such as accuracy and ROC.
